# Assigning values 

The operators used to assign values are:

<br>

|Operator | Example					| Equivalent		|
|:--------|:----------------|:--------------|
|=				| a = b 					|a = b					|'a' assigned value contained in 'b'						|
|+=				| a += b 					|a = (a + b)		|'a' assigned value of 'a' + 'b' operation			|
|-+				| a -= b 					|a = (a - b)		|'a' assigned value of 'a' - 'b' operation			|
|*=				| a *= b 					|a = (a * b)		|'a' assigned value of 'a' * 'b' operation			|
|/=				| a /= b 					|a = (a / b)		|'a' assigned value of 'a' / 'b' operation			|
|%=				| a %= b 					|a = (a % b)		|'a' assigned remainder of 'a' / 'b' operation	|
